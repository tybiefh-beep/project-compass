## 🎯 为什么选择 Project Compass？

### 1️⃣ **最快的上手体验**

```
其他工具：
1. 安装环境（npm / Python / Ruby）
2. 下载配置文件
3. 修改配置
4. 运行命令
5. 生成文件
⏱️ 总耗时：20-30 分钟

Project Compass：
1. 打开网页（或下载 HTML）
2. 输入 API Key
3. 对话回答 4 个问题
4. AI 自动生成
⏱️ 总耗时：5-8 分钟 ⚡
```

### 2️⃣ **独特的对话式交互**

```
对话式生成的优势：

❌ 传统方式：
- 手写 CLAUDE.md（很难知道该写什么）
- 或者用脚手架生成后手动修改

✅ Project Compass：
- 工具问你关键问题
- 你自然地表达想法
- AI 理解你的需求
- 自动生成专业的文档
```

### 3️⃣ **8+ 模型支持的灵活性**

```
支持的 AI 模型：

Claude (Anthropic)
- ✅ Opus 4.7 (质量最好)
- ✅ Sonnet 4.6 (平衡)
- ✅ Haiku 4.5 (快速)

GPT (OpenAI)
- ✅ GPT-4.1 (强大)
- ✅ GPT-4o (推荐)
- ✅ o3-mini (经济)

中国模型
- ✅ DeepSeek (便宜)
- ✅ 通义千问 (免费)
- ✅ 智谱 GLM (免费)

其他
- ✅ Gemini (Google 免费)
- ✅ Moonshot (便宜)
- ✅ 自定义 OpenAI 兼容接口

竞品对比：
- claude-code-templates: 仅支持 Claude
- html-anything: 主要 Claude，部分 Gemini
```

### 4️⃣ **纯本地隐私保护**

```
数据流向：

其他工具可能：
浏览器 → 第三方服务器 → AI API
            ↓
        (数据可能被记录)

Project Compass：
浏览器 → 直连 AI API
└── API Key 仅存本地浏览器
└── 无中间服务器
└── 完全隐私
```

### 5️⃣ **生成既然还包含 SKILL.md**

```
竞品输出：
- claude-code-templates: CLAUDE.md 只
- html-anything: HTML 文件

Project Compass: CLAUDE.md + SKILL.md

为什么 SKILL.md 重要？
- SKILL.md 定义了 AI 的可复用技能
- 让 Claude 知道何时自动执行
- 何时需要人工审核
- 让 AI 工作流更高效
```
